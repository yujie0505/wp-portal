# wp-portal

## android-sdk

### SETUP

```bash

# Set Environment Variables:
#
# add following informations into ~/.bashrc
#----------------------------------------------------------
#
# export ANDROID_HOME="$HOME/android-sdk/"
# export PATH="$HOME/android-sdk/platform-tools/:$PATH"
# export PATH="$HOME/android-sdk/tools/bin/:$PATH"

$ vi ~/.bashrc
$ source ~/.bashrc

# Accept sdk Licenses

$ sdkmanager --licenses

```
